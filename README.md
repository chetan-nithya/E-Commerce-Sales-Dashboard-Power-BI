# E-Commerce-Sales-Dashboard-Power-BI
The project focuses on transforming raw transactional data into meaningful business insights through data cleaning, modeling, and visualization techniques.

# Project Overview
This project presents an interactive E-Commerce Sales Dashboard built using Power BI. The dashboard provides comprehensive insights into revenue, customer behavior, product performance, and discount analysis.
The objective of this project is to transform raw sales data into meaningful business insights using data visualization techniques.

# Business Objectives
* Analyze total revenue and order trends
* Identify top-performing cities and products
* Understand category-level contribution
* Evaluate discount impact on revenue
* Track monthly sales performance
* Identify top 5 revenue-generating customers

# Key KPIs
* Final Revenue: 69.53M
* Total Customers: 552
* Total Orders: 570
* Discount Amount: 93.86
